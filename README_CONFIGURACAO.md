# Lar Buceles Command Center v16 Master Free

Versão final gratuita para GitHub Pages + Firebase Spark.

## O que esta versão inclui

- Mobile premium para iPhone, iPad, Android, tablet, notebook e PC.
- Menu inferior limpo no celular.
- Botão flutuante de ações rápidas.
- Filtros avançados por texto, responsável, status/prioridade e data.
- Templates expandidos para rotina diária, semanal, mensal, contas, compras, estoque, manutenção, documentos, orçamento, espiritualidade, viagem, emergência e reforma/mudança.
- Saúde do Lar ponderada por contas, tarefas, orçamento, estoque, agenda, documentos, manutenção, projetos, espiritualidade e reunião semanal.
- Relatórios profissionais com PDF pelo navegador, CSV, backup completo e backup mensal.
- Reunião semanal mais completa.
- Painel apresentação/TV.
- Privacidade com PIN local e bloqueio automático por inatividade.
- Integração Google Agenda em lote para datas pendentes.
- Offline/cache local e PWA.

## Como publicar

1. Envie todos os arquivos desta pasta para o repositório GitHub Pages `docx7/lar-buceles`.
2. Publique novamente o conteúdo de `firebase.rules` em Firebase > Firestore Database > Rules.
3. Abra: https://docx7.github.io/lar-buceles/
4. Se carregar versão antiga, use Ctrl+F5 ou limpe o cache do navegador/service worker.

## Não precisa pagar nada

Não ative Blaze. Não use Cloud Functions. Não use Firebase Cloud Messaging. Esta versão foi feita para rodar somente com GitHub Pages + Firebase Spark dentro da cota gratuita.
